AlexMDB

This web app provides users access to information about different movies, directors, and genres. Users can sign up, edit personal info, and create a list of their favorite movies.

REST API
ENDPOINT DESCRIPTION METHOD PARAMETERS REQUESTBODY RESPONSEBODY
/movies Returns all movies GET None None JSON [ { "Genre": { Name, Description }, "Director": { Name, Bio, Birthyear, Deathyear }, "Actors": [Actors], "ID": ID, "Title": Title, "Description": Description, "ImagePath": Link, "Featured": Boolean }, ]
/Movies/:Title/ Returns specific movie by title GET :Title None JSON, { "Genre": { Name, Description }, "Director": { Name, Bio, Birthyear, Deathyear }, "Actors": [Actors], "ID": ID, "Title": Title, "Description": Description, "ImagePath": Link, "Featured": Boolean }
/Movies/Genre/:Name Returns specific movie's genre by name GET :Name None JSON, "Genre": { Name, Description }
Movies/Director/:Name/ Returns a director by name GET :Name None JSON, "Director": { Name, Bio, Birthyear, Deathyear },
/users/ Returns all users GET None None JSON, [ { "Username": Username "Password": Password, "Email": Email, "Birthday": Birthday "FavoriteMovies": [FavoriteMovies] }, ]
/users/:Username Returns User by Username GET None None JSON, { "_id": ID, "Username": Username "Password": Password, "Email": Email, "Birthday": Birthday, "FavoriteMovies": [FavoriteMovies] }
/users/ Registers a new username POST None JSON, { "Username": Username "Password": Password, "Email": Email, "Birthday": Birthday } JSON, { "_id": ID, "Username": Username "Password": Password, "Email": Email, "Birthday": Birthday, "FavoriteMovies": [FavoriteMovies] }
/users/:Username/Movies/:MovieID Add movie to FavoriteMovies by a movie's ID POST :Username, :MovieID None JSON, { "_id": ID, "Username": Username "Password": Password, "Email": Email, "Birthday": Birthday, "FavoriteMovies": [FavoriteMovies] }
/users/:Username/Movies/:MovieID Add movie to my movie list POST :Username, :MovieID None JSON, { "_id": ID, "Username": Username "Password": Password, "Email": Email, "Birthday": Birthday, "FavoriteMovies": [FavoriteMovies] }
/users/:Username/ Update username PUT :Username JSON, { "Username": Username "Password": Password, "Email": Email, "Birthday": Birthday } JSON { "_id": ID, "Username": Username "Password": Password, "Email": Email, "Birthday": Birthday, "FavoriteMovies": [FavoriteMovies] }
/users/:Username/ Delete user DELETE :Username None None
/users/:Username/Movies/:MovieID Remove movie from my movie list DELETE :Username, :MovieID None None
/users/:Username Delete User DELETE :Username, None None
