AlexMDB

This web app provides users access to information about different movies, directors, and genres. Users can sign up, edit personal info, and create a list of their favorite movies.

REST API
Endpoint Description Method Type Parameters Request Body Response Body
/movies Returns all movies GET none none JSON
/movies/:title/ Returns specific movie by title GET :title none JSON
/movies/:title/:genre/ Returns specific movie's genre by title GET :title, :genre none JSON
/directors/:name/ Returns a director by name GET :name none JSON
/users/ Registers a new username POST :title JSON JSON
/users/:username/ Update username PUT :username JSON JSON
/users/:username/ Delete user DELETE :username none JSON
/users/:username/my-movies/ Add movie to my movie list POST :username JSON JSON
/users/:username/my-movies/ Remove movie from my movie list DELETE :username JSON JSON
